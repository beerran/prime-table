Dynamic tables made easier! Use the configurable PrimeTable classes to easily configure and set up new data tables

PrimeTable is an Angular wrapper for the PrimeNg component suite. Achieve features such as sorting, row expanding, data drilldown, row selection, filtering and more with minimal configuration.

Installation

To install PrimeTable into an existing Angular application, simply run:

npm install prime-table --save
